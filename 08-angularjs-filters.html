Program 8: AngularJS Filters Implementation

1. Uppercase Filter

Syntax: {{ '{' }}{{ '{' }} expression | uppercase {{ '}' }}{{ '}' }}
Input:
Original: {{ text1 }}
Output (Uppercase):
{{ text1 | uppercase }}

2. Lowercase Filter

Syntax: {{ '{' }}{{ '{' }} expression | lowercase {{ '}' }}{{ '}' }}
Input:
Original: {{ text2 }}
Output (Lowercase):
{{ text2 | lowercase }}

3. Currency Filter

Syntax: {{ '{' }}{{ '{' }} amount | currency : symbol : fractionSize {{ '}' }}{{ '}' }}
Enter Amount: Select Currency:
Formatted Currency:
Default: {{ amount | currency }}
Custom: {{ amount | currency : currencySymbol }}
No Decimals: {{ amount | currency : currencySymbol : 0 }}

4. Number Filter

Syntax: {{ '{' }}{{ '{' }} number | number : fractionSize {{ '}' }}{{ '}' }}
Enter Number:
Original: {{ numberValue }}
Formatted Number:
Default: {{ numberValue | number }}
2 Decimals: {{ numberValue | number : 2 }}
4 Decimals: {{ numberValue | number : 4 }}
No Decimals: {{ numberValue | number : 0 }}

5. Date Filter

Syntax: {{ '{' }}{{ '{' }} date | date : format : timezone {{ '}' }}{{ '}' }}
Current Date/Time Formats:
Full Date: {{ currentDate | date : 'fullDate' }}
Short Date: {{ currentDate | date : 'shortDate' }}
Medium Date: {{ currentDate | date : 'mediumDate' }}
Long Date: {{ currentDate | date : 'longDate' }}
Short Time: {{ currentDate | date : 'shortTime' }}
Medium Time: {{ currentDate | date : 'mediumTime' }}
Custom (dd/MM/yyyy): {{ currentDate | date : 'dd/MM/yyyy' }}
Custom (MMM dd, yyyy HH:mm): {{ currentDate | date : 'MMM dd, yyyy HH:mm' }}
Day & Month: {{ currentDate | date : 'EEEE, MMMM dd' }}

6. LimitTo Filter

Syntax: {{ '{' }}{{ '{' }} array | limitTo : limit : begin {{ '}' }}{{ '}' }}
Limit Number:
Full Text: {{ sampleText }}
Limited Text:
{{ sampleText | limitTo : limitNumber }}...
Array (First {{ limitNumber }}): {{ numbers | limitTo : limitNumber }}

7. OrderBy Filter

Syntax: ng-repeat="item in array | orderBy : expression : reverse"
Sort by:
Name Age Department Salary
{{ person.name }} {{ person.age }} {{ person.department }} {{ person.salary | currency }}

8. Filter (Search) Filter

Syntax: ng-repeat="item in array | filter : searchText"
Search Products:
Product Name Category Price Stock
{{ product.name }} {{ product.category }} {{ product.price | currency }} {{ product.stock }}

Results: {{ (products | filter : searchProduct).length }} of {{ products.length }} products

9. JSON Filter

Syntax: {{ '{' }}{{ '{' }} object | json {{ '}' }}{{ '}' }}
Object to JSON: 
{{ sampleObject | json }}

10. Custom Filter (Reverse Text)

Custom Filter: Created using app.filter('reverse', function() {...})
Input Text:
Original: {{ reverseText }}
Reversed Text:
{{ reverseText | reverse }}

11. Chaining Multiple Filters

Syntax: {{ '{' }}{{ '{' }} expression | filter1 | filter2 | filter3 {{ '}' }}{{ '}' }}
Input:
Original: {{ chainText }}
Chained Filters:
Uppercase: {{ chainText | uppercase }}
Uppercase + LimitTo 10: {{ chainText | uppercase | limitTo : 10 }}
Lowercase + Reverse: {{ chainText | lowercase | reverse }}

12. Using Filters in Controller

Results from Controller:
Uppercase: {{ controllerResult.upper }}
Currency: {{ controllerResult.currency }}
Date: {{ controllerResult.date }}
